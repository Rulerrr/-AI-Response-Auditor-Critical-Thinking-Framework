## The Core: The Human Logic Handbook
The heart of this project is the **[Handbook of Sophisms and Paradoxes](./Sofizm-eng.html)**. It serves as a logical baseline independent of AI hallucinations. 

Before you trust the "vibe" of an answer, use this handbook to identify:
* **Systemic Paradoxes:** (Braess, Goodhart, Jevons) to see how the solution might break the system.
* **Logical Fallacies:** (Post Hoc, Circular Reasoning, Base Rate Fallacy) to dismantle the "illusion of logic".
* **Semantic Leakage:** To catch when cultural tropes replace evidence-based reasoning.

## The Workflow: Augmenting Human Agency
We do not delegate the responsibility for truth to the machine. We use the **[PROMPT.md](./PROMPT.md)** only to *augment* our own auditing process.

1. **Observe:** Identify the "manic enthusiasm" or "safe" diplomatic answer in the LLM output.
2. **Challenge:** Apply the Auditor Directive to force the model into an "intellectually brutal" state.
3. **Verify:** Use the Human Logic Handbook to judge if the revised response has moved from "satisfying" to "logically sound".
